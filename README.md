# 🤖 Robot Following Cursor Landing Page

A modern and interactive landing page featuring a 3D robot that follows the user's cursor. Built with smooth animations and immersive visual effects to create an engaging browsing experience.

**🗓️ Project created on:** June 21, 2026

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
* Spline Viewer
* AOS (Animate On Scroll)
